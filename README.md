1. Run the project through login.
2. Click sign up on the button and make an account.
3. CLick back to login and put your info.
4. (The homepage will be empty at first because you havent added any recipes.)
5. First you need ingredients to make a recipe so click the ingredients page button
6. Click new ingredient button at the bottom.
7. Fill out the name and macros.
8. Add as many ingredients as you want for your recipe.
9. Click back to the home page
10. Click create recipe at the bottom
11. ***This is important*** Enter recipe information then CREATE recipe first before adding ingredients. If you dont then an error message will pop up (generated by me so my code didnt fail yet...) and it will tell you to create recipe first.
12. After creating the recipe, choose your ingredients and click add to add the ingredients to the recipe.
13. Go to home page and you can view or edit the recipes.
